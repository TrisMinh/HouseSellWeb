# Rule: Review Memory

## Quy tắc bắt buộc

- Mỗi lần hoàn thành công việc, phải cập nhật file review để lưu “bộ nhớ vận hành”.

## File chuẩn

- `review/REVIEW.md`

## Nội dung bắt buộc trong mỗi lần cập nhật

- Việc đã làm.
- File đã thay đổi.
- Kết quả kiểm tra/verification.
- Rủi ro còn lại.
- Bài học rút ra.
- Bước tiếp theo đề xuất.
